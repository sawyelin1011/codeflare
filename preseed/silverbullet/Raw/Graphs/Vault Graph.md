# Vault Graph

Interactive force-directed visualization of every concept and document the vault-extract agent has ingested. Regenerated by the agent on each background extraction tick.

[Open vault graph viz](vault-graph.html)

Source data: `graphify-out/vault-graph.json` (hidden from the treeview by design — derived build output).

## What you'll see

- One node per page in the vault, plus one node per `[[wikilink]]` concept.
- Edges connect pages to the concepts they cite.
- Node colour reflects the graphify community label; size reflects centrality.
